Welcome to Quick-Send support. We're here to help you get the most out of our file sharing app.

Getting Started

How to Share Files

  1. Open Quick-Send on your device
  2. Tap the file selection area
  3. Choose the file you want to share
  4. Tap "Generate Link"
  5. Copy or share the generated link

File Types Supported

Quick-Send supports most common file types including documents, images, videos, and audio files.

Subscription Plans

Free Plan

  • 100MB file size limit
  • 7-day file expiry
  • 10 uploads per month
  • Basic file sharing

Pro Plan

  • 1GB file size limit
  • 30-day file expiry
  • 100 uploads per month
  • No ads

Business Plan

  • 5GB file size limit
  • 90-day file expiry
  • 1000 uploads per month
  • No ads

Common Issues

File Upload Failed

Check your internet connection and ensure the file size is within your plan's limits.

Link Not Working

Links expire automatically based on your subscription plan. Generate a new link if needed.

Subscription Issues

For subscription-related questions, please contact Apple Support through your device settings.

Privacy & Security

Your privacy is important to us. All uploaded files are automatically deleted after the expiry period. We do not store or access your personal files beyond what's necessary to provide our service.
